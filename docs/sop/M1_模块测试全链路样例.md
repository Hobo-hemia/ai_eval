# M1（协议架构）自动化评测 SOP（Cursor）

## 目标

通过统一命令完成 M1 评测：

1. `ai_eval init`：初始化候选模型目录
2. `ai_eval run`：执行 `Phase1 -> Phase2 -> Phase3`
3. `ai_eval clear`：清理评测产物

---

## 评测目标（M1）

M1 聚焦“从抽象 PRD 提炼协议主链路并演进已有 proto”的能力，要求模型：

- 改造既有 `api.proto`（含接口重命名与字段演进）
- 支撑 5 个产品功能点
- 新增接口数 `< 5`，体现后端抽象与收敛

---

## 前置条件

```bash
cd /path/to/ai_eval
cursor-agent login
go install .
```

---

## Step 1：初始化目录

```bash
ai_eval init --models "gpt-5.2,sonnet-4.5" --modules "m1"
```

---

## Step 2：执行单模型评测

```bash
ai_eval run --module m1 --model "gpt-5.2" --judge-model "opus-4.6"
```

---

## 结果产物

- `eval_records/<model_dir>/m1_arch/m1_result.proto`
- `eval_records/<model_dir>/m1_arch/m1_build.log`
- `eval_records/<model_dir>/m1_arch/m1_test.log`
- `eval_records/<model_dir>/m1_arch/score.json`
